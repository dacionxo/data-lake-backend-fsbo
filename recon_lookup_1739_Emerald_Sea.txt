Result: No homeowner data from recon-ng
--------------------------------------
Recon-ng does not include any module that looks up property owner or resident
information by street address. The framework is built for:

  - Domain/company/contact reconnaissance (e.g. domains -> contacts, companies -> contacts)
  - Location-related modules in the marketplace only:
    - geocode: address -> latitude/longitude (requires Google API key)
    - reverse_geocode: coordinates -> address
    - locations-pushpins: Flickr/Shodan/Twitter/YouTube media near a location

None of these produce owner name, phone, or email for an address.
